Superuser status to create, access and edit files within the htdocs file.

April 2022

To become a superuser so you can access the htdocs file, type the following into the terminal window:

sudo su
Enter you password

THis shows you are a superuser
root@maurizio-Inspiron-11-3162:/home/maurizio#
The 'root@' and 'hash' at the very beginning and end signifies you are a superuser.

As a superuser I am able to create, edit and save contents within htdocs

Creating and opening a file in terminal

  • touch start.php
  • bluefish start.php (This shouts a warning at you WARNING **: 11:09:26.133: no config.........., but opens the file all the same.)
